Overview

Anynines offers a number of sample applications to get you started. These samples are written in a variety of programming languages. Here they are.

Sample Applications

You can find most of our sample applications on our github anynines account at https://github.com/anynines/.

You will need to signup with Anynines before running any of these samples.

Node.js

  • Simple Node App NodeJS , no services; A simple nodejs app to test a deployment environment.

  • Nodejs Mongodb Example NodeJS , mongodb; A simple quotation database application using node.js, MongoDB and express.js.

Ruby

  • Paperclip Demo - Rails,PostgreSQL service, Swift service; Application allows users to manage a list of friends.

  • Rails4 Carrierwave Example - Rails,PostgreSQL, Swift service; Rails 3.2.18 & Carrierwave & a9s Swift Service & PostgreSQL. A small application that demonstrates the work of these programs together.

  • A9S Swift - a support module for paperclip and carrierwave.

  • Ruby Sinatra Example App - Sinatra, no services.

  • Simple Rails App - Rails, Ruby 1.9.3, 2.0.0, JRuby, MySQL services; It can create, delete and display posts. It is just a proof of concept.

  • Ruby Rails Example 2 - another one.
